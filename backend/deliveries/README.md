# WorkSphere Deliveries

## Structure

| Directory/File | Description |
|----------------|-------------|
| `deployment/` | Multi-cloud IaC (AWS, Azure, GCP, K8s, Terraform, CI/CD) |
| `docker/` | Docker Compose + service Dockerfiles |
| `scripts/` | Automation & fix scripts |
| `github/` | GitHub Actions workflows |
| `frontend-dist/` | Built frontend distribution |
| `backend-dist/` | Built backend distribution |
| `vercel.json` | Vercel deployment config |
| `run_all.bat` | Universal launcher |
| `package.json` | Root monorepo package |
| `.gitignore` | Git ignore rules |
| `.env.example` | Environment template |
